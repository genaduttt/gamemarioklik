Fitur :
Terdapat 5 button
1. Button “Klik!” merupakan button untuk menambah coins sebanyak 1 coins/klik. button ini juga untuk memulai permainan, 
karena button lain belum bisa digunakan karena coins player yang belum cukup untuk dibeli.
2.Button “mushroom” berguna untuk menambah +1 coins setiap player menekan button “Klik!”. contoh: coins player masih 0 
disaat player menekan 1x button “Klik!”  coins akan meningkat menjadi 2 coins.   jika player membeli  “mushroom” lagi 
coins akan bertambah +1 coins lagi, tetapi harga akan meningkat.
3. Button “Star” hampir sama konsepnya dengan button “mushroom” yang membedakan hanya coins menambah sebanyak +5 coins 
setiap klik dan harga jauh lebih mahal.
4. Button “2x multiplier” berguna untuk menggandakan coin yang ditekan. Contoh: player beli star 1+5 = 6, lalu player 
beli 2x multiplier jadi 6x2 = 12 coin per click.
5. Button “Auto cliker” berguna untuk meningkatkan coins tanpa harus menekan button “Klik!”.
6. Apabila kita mengklik tombol "Klik!" maka gambar karakter akan melompat sekaligus akan menghasilkan suara
7. Apabila kita mengklik tombol "2x Multiplier","Auto-cliker","Mushroom (+1 coins)", dan "Star (+5 coins)" maka akan
menghasilkan efek suara

Rule :
1. Untuk bisa menggunakan button lain, player harus mengumpulkan coins sesuai harga button tersebut. Jika coins tidak cukup 
akan menampilkan “ You don’t have enough money to buy this item!”
2. Setiap kita membeli "Mystery Box" dan "Foods" maka harganya akan semakin naik
